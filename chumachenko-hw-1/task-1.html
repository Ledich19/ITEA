Формулы и свойства степеней

Свойства и формулы степеней используются при сокращении и упрощении сложных выражений, при  решении уравнений и неравенств. Свойства степеней можно использовать совместно с  таблицей степенейи  таблицей умножения

a1 = а, a 0  = 1 (a ≠ 0), a-n = 1/an.
  • aman= am+n ;
  • am/an= am-n ;
  • (ab)n= anbn ;
  • (am)n= amn ;
  • (a/b)n= an/bn ;
